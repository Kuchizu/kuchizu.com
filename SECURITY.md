# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

- **Email:** legolashacker0@gmail.com
- **GitHub:** [Open a private security advisory](https://github.com/Kuchizu/kuchizu.com/security/advisories/new)

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

I will respond within 48 hours.

## Scope

This policy applies to:

- kuchizu.com website
- API endpoints (`/api/*`)
- Docker images in `ghcr.io/kuchizu/*`

## Out of Scope

- Third-party services (Spotify, Steam, GitHub APIs)
- Umami analytics instance
- Denial of Service attacks
